1. Installation de Python

Télécharger Python 3.12.5

  • Téléchargez Python 3.12.5
  • Lors de l'installation, assurez-vous de cocher l'option "Ajouter Python au PATH".

Créer et Activer un Environnement Virtuel

  • Créer l'environnement virtuel dans le répertoire du projet Parking Car Management :
    • python -m venv .venv
  • Activer l'environnement virtuel :
    • .venv\Scripts\Activate.ps1
  • Installer les dépendances à partir du fichier requirements.txt :
    • pip install -r requirements.txt

2. Installation de PostgreSQL et pgAdmin

  • Installer PostgreSQL version 16 : Téléchargez PostgreSQL
  • Installer pgAdmin version 4 (8.11) : Téléchargez pgAdmin 4

3. Configuration de PostgreSQL

Ouvrir pgAdmin et Créer une Base de Données

  • Ouvrez pgAdmin et connectez-vous à votre serveur PostgreSQL.
  • Créez une nouvelle base de données avec les paramètres suivants :
    • DATABASES = {
    'default': {
        'NAME': 'mytestdb',
        'USER': 'postgres',
        'PASSWORD': 'root',
        'HOST': 'localhost',
        'PORT': '5433',
    }
}

4. Gestion du Projet Parking Car Management avec Django

Activer l'environnement et Démarrer le Serveur

  • Ouvrir un terminal dans le répertoire du projet et activer l'environnement :
    • .venv\Scripts\Activate.ps1
  • Démarrer le serveur Django :
    • python manage.py runserver
  • Si tout fonctionne bien, fermez le serveur avec (CTRL + C).

Migrations de la Base de Données

  • Créer les migrations pour le projet Parking Car Management :
    • python manage.py makemigrations
  • Appliquer les migrations à la base de données :
    • python manage.py migrate

Redémarrer le Serveur

  • Enfin, redémarrez le serveur avec :
    • python manage.py runserver
